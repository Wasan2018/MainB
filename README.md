# ล่าสุด


##### ที่ทำได้

  - ค้นหาไฟล์ใน SD Card
  - เปิดอ่านไฟล์
  - เลื่อนบรรทัด

##### ที่กำลังทำ

  - โหมดพิมพ์ (เหมือน notepad)
  - โหมดอ่านไฟล์ ยังย้อนกลับมาอ่านไม่ได้
  - ยังเปิดและดูไฟล์ในโฟลเดอร์ไม่ได้
